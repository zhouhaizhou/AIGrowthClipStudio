AI Growth Clip Studio

增长素材审核台
📖 使用指南 / 功能说明（点击展开 / 收起）

本工具把一条长视频自动拆成多版本（推荐流 / 广告 / 会员 / 社媒）短视频增长素材：AI 转写字幕、识别高光片段、生成标题 / 封面文案 / 推荐语；你在此预览、微调、审核上线，并回流投放数据做效果分析。

① 基本流程（四步）

1
创建任务 —— 展开下方「＋ 创建任务」，填视频地址与目标参数后提交。
2
等待处理 —— 任务由后台 worker 处理（转写 → 选高光 → 生成文案 → 切片）。注意：本控制台不会自动处理任务，必须有 worker 进程在运行。在「任务」区点「刷新」，直到状态变 succeeded。
3
审核素材 —— 点任务行，下方出现素材卡。可直接修改标题 / 封面文案 / 推荐语，再点「通过」或「驳回」。
4
效果回流 —— 点素材上的「模拟埋点」上报投放数据，再到「效果分析」点「刷新」查看聚合数据与优化建议。

② 区块功能

  • ＋ 创建任务：发起一个视频拆条任务。
  • 任务：所有任务及状态 / 进度；点击某行即选中并在下方加载它的素材。
  • 素材：当前选中任务产出的多版本短视频，逐条预览 + 审核。
  • 效果分析：全局（跨所有任务）投放数据聚合（总览 / 按场景 / 按高光类型）+ AI 优化建议。

③ 按钮行为

  • 通过 / 驳回：会把你当前编辑后的标题 / 封面 / 推荐语一起入库，并把素材状态改为 approved / rejected。
  • 模拟埋点：上报一条模拟投放数据（曝光固定 50，点击 / 播放 / 完播 / 分享随机），用于演示效果回流。
  • 刷新：任务区刷新任务列表；分析区重新拉取聚合数据。

④ 状态徽章

  • 任务：queued 排队 · running 处理中 · succeeded 完成 · failed 失败
  • 素材：pending_review 待审核 · approved 已通过 · rejected 已驳回

⑤ 创建任务字段说明

  • sourceVideoUrl：视频地址，需 file:// 绝对路径（暂不支持 http 直链），例 file:///abs/path/video.mp4。
  • sourceContentType：内容类型，video（单条视频）或 episode（剧集 / 长视频）。
  • targetScenarios：目标投放场景；AI 会据此为每条素材分配场景。
  • targetDurations × targetAspectRatios：每个高光片段按所选「时长 × 比例」各切一版。
  • targetLanguages：字幕 / 文案语言（默认 zh-CN）。
  • clipCount：期望产出的高光片段数量。
